1 Session 1

Setting up the Python Development Environment

Introduction

This worksheet will guide you through setting up your computer for Python programming. We will create a GitHub account, choose a suitable programming environment, and install the necessary tools.

The Role of Each Tool

The ecosystem helps beginners see the big picture before they start clicking Install.

  1. Python Interpreter is the engine that reads and executes your code.

  2. The IDE (Integrated Development Environment) is your digital workshop for writing, debugging, and organising code (e.g., VS Code or PyCharm).

  3. Git & GitHub are the mechanisms for storing locally and in the cloud. Git tracks changes locally, while GitHub hosts those changes online for backup and collaboration.

Verification Checklist

Ensure the installation was successful to reduce troubleshooting time during the actual labs.

  1. To check if Python has been installed correctly, open your terminal/command prompt and type:
python --version or python3 --version

To confirm the installation was successful and to check the version of Python installed.

  1. To check Git has been installed correctly, open your terminal/command prompt and type:

git –version

To confirm the installation was successful and to check the version of Git installed.

Understanding the Command Line Interface (CLI)

Since the worksheet mentions using the Command Line Interface (CLI) and terminal, a small tip on how to access it is helpful:

Windows

Use PowerShell, Command Prompt (cmd) or Git Bash after installing Git.

macOS

Use the Terminal app found in Applications > Utilities folder.

Why use Virtual Environments?

While the worksheet mentions Virtualised or Container Environments, you should review the documentation for Python Virtual Environments (venv), which is a standard industry practice for running your applications.

Note: Virtual environments allow you to create isolated spaces for different projects, preventing version conflicts between libraries.

Create a GitHub Account

  1. Open a web browser and navigate to https://github.com/.

  2. Click on Sign up if you intend to use GitHub for this course, then use a personal email address (I would suggest you use your college email address).

  3. Fill in the required information (username, email, password).

  4. Follow the on-screen instructions to verify your email address.

Choose your Development Environment

Consider the following factors when choosing your environment:

  1. Different operating systems (OS) can be used for this programming course. i.e, laptop or tablet running macOS, Linux, Android or Windows?

Popular editor options

Select an Integrated Development Environment (IDE):

The following are just some of the possible IDSs.

  1. Visual Studio Code (vscode), which is a Cross-platform, lightweight, extensible application

  2. PyCharm isa full-featured, Python-specific application.

  3. Jupyter Notebook and Google Colab are interactive tools for data exploration, visualisation, and narrative text in a single document.

Choose a Virtualised or Container Environment

If you only intend to set up your IDE for this course only, then you could install a Virtualised or Container Environment.

  1. Containerisation tools like Docker are used to isolate applications and ensure they run in a consistent environment.

  2. Virtualisation using an application like VirtualBox to create virtual machine environments that can run different operating systems.

Install your chosen IDE

Follow the specific installation instructions for your chosen environment.

Example: Installing Visual Studio Code on macOS or Windows.

  1. Download the installer from https://code.visualstudio.com/.

  2. Open the downloaded file and follow the on-screen instructions.

Install Git

What is Git? Git is an essential version control system for working with GitHub.

macOS

Install Git using Homebrew, using a bash Command Line Interface (CLI)

  1. brew install git

    1. Alternatively, download the installer from https://git-scm.com and follow the instructions.

Windows

Download the installer from https://git-scm.com/downloads and follow the instructions.

Configure the local copy of Git

Once you have installed git locally, you will need to set up some default configuration.

  1. Open your terminal (CLI) or command prompt.

  2. Configure Git with your name and email.

git config –global user.name “Your Name” git config –global user.email “your_email@example.com”  

Remember to replace placeholders “user.name” and “user.email” with your actual information.

Install Python

Most IDEs come with Python pre-installed or have options to install it.

  1. If you need a standalone Python installation, download it from https://www.python.org/ and follow the installation instructions.

Install Required Extensions (Optional)

Install extensions to enhance your IDE’s capabilities.

Visual Studio Code

  1. Python extension.

  2. Git or GitHub extension.

Test Your Setup

  1. Create a new Python file in your chosen environment.

  2. Write a simple Python script by using the sample Python code below:

  print("Hello, world!")

  1. Run the script.

  2. Create a new Git repository and push your code to GitHub.

Additional Tips

  1. Experiment by trying different environments to find what works best for you.

  2. Learn Git, and you will gain more value by taking the time to understand what and how to use its commands, as it is crucial for effective version control.

  3. Utilise online resources by researching the sites, as there are many tutorials and documentation available.

  4. Connect with other programmers for support and knowledge sharing by joining communities.

Note: This worksheet provides a general outline. Specific instructions may vary depending on your operating system, chosen environment, and software versions.

2 Session 2

Syntax, Variables, Types, and Operators

Create simple code scripts to perform basic arithmetic operations and variable manipulations.

Variable Declaration and Types:

  1. Declare two variables, a and b. Assign a the value 15 and b the value 12.

  2. Print the types of a and b using the type() function.

Basic Arithmetic Operations:

  1. Create a code script to perform an addition using (a + b) and print the result.

  2. Create a code script to perform a subtract (a - b) and print the result.

  3. Create a code script to multiply (a * b) and print the result.

  4. Create a code script to perform a division (a / b) and print the result.

Using Variables and Type Casting:

  1. Create a new variable c that stores the result of a divided by b. Make sure c is of type integer.

  2. Print the value and type of c.

  3. Now convert c to a float and print its new value and type.

Working with Strings:

  1. Declare a string variable message with the value The result of a divided by b is?

  2. Concatenate the message with the value of c, converted to a string and print the result.

Using Comparison Operators:

  1. Compare if a is greater than b then and print the result True or False.

  2. Check if a is equal to b and print the result True or False.

3 Additional instructions

To enhance clarity, present the code and its resulting output together. Each code interaction should be accompanied by documentation of actions taken and the expected output.

This will help you understand the relationship between the code and its output.

For example, after writing a code snippet, you can immediately follow it with a comment or a print statement that describes what the output should be.

This way, you can easily verify if your code is working as intended by comparing the actual output with the expected output.

Example to demonstrate the code and output

Python code:


# Define a and b.
a = 5
b = 3
myout = (a+b) # Add a + b and store to myout.
print("The sum of a + b equals to", myout) # Send to standard out.

Output:

The sum of a + b equals 8
