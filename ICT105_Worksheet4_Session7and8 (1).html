Functions are components in Python programming, defined for reusable code blocks that perform specific tasks. They allow you to break down complex problems into smaller, manageable units, making your code more organised, efficient, and easier to maintain.

1.2.2 (2) Read a CSV file into a dictionary.

A basic example of a Comma-Separated Values (CSV) student.csv file being loaded using pandas.

Python code (2):

'''
Read the CSV file into a DataFrame.
The pandas library is used to read the CSV file and store it in a DataFrame for easier manipulation.
'''
import pandas as pd

'''
Assuming the CSV file is located in the "WIP" directory and has no header row.
The columns are expected to be in the order: StudentID, StudentFirstname, StudentLastname, Email.
If the CSV file has a header row, remove the "header=None" argument.
'''

# df = pd.read_csv("WIP/students.csv", header = None)
df = pd.read_csv("students.csv", header = None)

# Create a dictionary to store the data
# The dictionary will use student IDs as keys and a list of [first name, last name, email] as values.

student_details = {}

# Iterate over the DataFrame rows
# The iterrows() function allows us to loop through each row of the DataFrame as (index, row) pairs.

for index, row in df.iterrows():
    student_id = row[0] # Assuming student ID is in the first column.
    student_firstname = row[1] # Assuming first name second column.
    student_surname = row[2] # Assuming surname in the third column.
    student_email = row[3] # Assuming email in the fourth column.

'''
Add the student data to the dictionary
The student ID is used as the key, and the value is a list containing the first name, surname, and email.
The if statement checks if the student ID is not already in the dictionary to avoid overwriting existing entries.
''' 
if student_id not in student_details:
    student_details[student_id] = [student_firstname, student_surname, student_email]

# Print the resulting dictionary entries
for student_id, details in student_details.items():
    print(f"Student Details: {student_id} {details[0]} {details[1]} {details[2]}")

1.2.3 (3) GUI user input example with alternative output methods.

This code implements a simple Student Information application using Tkinter.

Python code (3):


# Import the Tkinter library for creating the GUI application.    

import tkinter as tk
from tkinter import messagebox, filedialog

#  Define the StudentApp class to encapsulate the functionality of the application.

class StudentApp:
    
    #   The __init__ method initialises the application, setting up the main window and input fields.
    def __init__(self, root):
        self.root = root
        self.root.title("Student Information Input")
        self.students = []

        # Input form labels and entry fields
        tk.Label(root, text="Student Number:").grid(row=0, column=0, padx=5, pady=5)
        tk.Label(root, text="First Name:").grid(row=1, column=0, padx=5, pady=5)
        tk.Label(root, text="Last Name:").grid(row=2, column=0, padx=5, pady=5)

        # Entry fields for student number, first name, and last name.  
        self.student_number = tk.Entry(root)
        self.first_name = tk.Entry(root)
        self.last_name = tk.Entry(root)

        # Position the entry fields in the grid layout.      
        self.student_number.grid(row=0, column=1, padx=5, pady=5)
        self.first_name.grid(row=1, column=1, padx=5, pady=5)
        self.last_name.grid(row=2, column=1, padx=5, pady=5)

        # Buttons to add a student and show the list of students.  The buttons are linked to their respective methods.
        tk.Button(root, text="Add Student", command=self.add_student).grid(row=3, column=0, padx=5, pady=5)
        tk.Button(root, text="Show Students", command=self.show_students).grid(row=3, column=1, padx=5, pady=5)
        tk.Button(root, text="Save to File", command=self.save_to_file).grid(row=4, column=0, columnspan=2, padx=5, pady=5)

    '''
    The add_student method retrieves the input from the entry fields, validates it, and adds it to the students list. 
    It also provides feedback to the user through message boxes.
    '''
    def add_student(self):
        
        # Adds a student to the list.
        student_number = self.student_number.get()
        first_name = self.first_name.get()
        last_name = self.last_name.get()

        # Check if all fields are filled before adding the student to the list.  If any field is empty, a warning message is shown.
        if student_number and first_name and last_name:
            self.students.append((student_number, first_name, last_name))
            messagebox.showinfo("Success", "Student added successfully!")
            self.clear_entries()
        else:
            messagebox.showwarning("Incomplete Data", "Please fill in all fields.")

    # The clear_entries method clears the input fields after a student is added successfully.
    def clear_entries(self):
        
        #Clears the input fields.
        self.student_number.delete(0, tk.END)
        self.first_name.delete(0, tk.END)
        self.last_name.delete(0, tk.END)

    #  The show_students method opens a new window to display the list of students.  If there are no students added, it shows an informational message.
    def show_students(self):

        # Opens a new window to display the student list.
        if not self.students:
            messagebox.showinfo("No Data", "No students added yet.")
            return

        '''
        Creates a new window to display the student list.  
        A Text widget is used to show the details of each student, and a button is provided to save the list to a file.
        '''
        display_window = tk.Toplevel(self.root)
        display_window.title("Student List")

        text_area = tk.Text(display_window, width=50, height=20)
        text_area.pack(padx=5, pady=5)

        '''
        Loop through the students list and insert their details into the Text widget.  
        Each student's information is formatted and displayed on a new line.
        '''
        for student in self.students:
            text_area.insert(tk.END, f"Student Number: {student[0]}, "
                                         f"First Name: {student[1]}, "
                                         f"Last Name: {student[2]}\n")

        # Button to save the student list to a file, linked to the save_to_file method.
        tk.Button(display_window, text="Save to File", command=self.save_to_file).pack(pady=5)

    '''
    The save_to_file method allows the user to save the list of students to a text file.  
    It uses a file dialog to choose the save location and handles the file writing process.
    '''
    def save_to_file(self):
        
        # Saves the student list to a file.
        if not self.students:
            messagebox.showinfo("No Data", "No students to save.")
            return
        
        file_path = filedialog.asksaveasfilename(
            defaultextension=".txt",
            filetypes=[("Text Files", "*.txt"), ("All Files", "*.*")]
        )

        # If a file path is selected, the method writes the student details to the file.
        if file_path:
            with open(file_path, "w") as file:
                for student in self.students:
                    file.write(f"Student Number: {student[0]}, "
                               f"First Name: {student[1]}, "
                               f"Last Name: {student[2]}\n")
            messagebox.showinfo("Saved", "Student data saved successfully!")

# Main application setup
if __name__ == "__main__":
    root = tk.Tk()
    app = StudentApp(root)
    root.mainloop()

2.1 Object-Orientated Programming (OOP) Working with Classes & Methods

2.1.1 Introduction

Imagine a blueprint for creating houses. This blueprint outlines the structure, materials, and layout of a house. In programming, a class is like this blueprint, defining the structure and behaviour of objects. A method is a function that defines a class’s behaviour.
