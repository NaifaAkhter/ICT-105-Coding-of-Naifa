1.1 Working with Lists, Tuples and Loops

1.1.1 Introduction

Lists, tuples, and loops are fundamental concepts in Python programming that allow you to work with data collections efficiently. Lists are ordered sequences of elements that can be modified, while tuples are immutable sequences that cannot be changed after creation. Loops are control flow statements that enable you to repeat actions multiple times, making them invaluable for tasks that involve iteration over data structures. Understanding these concepts will provide a solid foundation for building more complex Python applications.

5.3 Creating a Course Information Retrieval System

  1. Create a Python script to retrieve data from a list, using sample data from the units table [course_id, course_name, department, prerequisites].

  2. Store the course information from the provided table units in a suitable data structure using a list.

  3. Prompts the user to enter a course ID.

  4. Based on the entered course ID, the course name, department, and prerequisites are retrieved and displayed.

  5. Handles cases where the entered course ID is not found.

5.3.1 Instructions

  • Data Structure - Use a multidimensional list to store course information. The course ID is the key, and the course details (Course Name, Department, and Prerequisites) are stored as values.

  • User Input - The user is prompted to enter a course ID.

  • Conditional Statement - One or more if-elif-else statements are needed to check if the entered course ID is in the list.

  • Information Retrieval - If the course ID is found, the corresponding course information is accessed using the index key and printed.

  • Error Handling - If the course ID is not found, an error message is displayed.

5.4 Additional Considerations

  • Loops - For a larger dataset, consider using a for loop to iterate over the list and find the matching course.

  • Input Validation - Implement input validation to ensure the user enters a valid integer for the course ID.

  • User Interface - Enhance the user interface by providing more informative messages and options.
